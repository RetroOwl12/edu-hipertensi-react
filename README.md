# Edu Hipertensi

Edu Hipertensi adalah website edukasi yang menyediakan informasi
tentang hipertensi (tekanan darah tinggi) untuk membantu masyarakat
memahami, mencegah, dan mengelola kondisi tersebut.

Project ini dibuat sebagai aplikasi frontend berbasis React dengan
fitur manajemen artikel oleh admin.
